# Harmony Pattern Knowledge Base

Read this before writing a new reharmonization pattern. This is a duplicate-detection index, not final website prose.

## Duplicate Check Procedure

For every new idea, compare:

- inserted chord or chord family
- target chord
- original slot in the progression
- bass motion
- inner-voice motion
- emotional effect
- glossary terms
- common variants

If a new idea matches an existing entry in at least two core signals, suggest updating the existing post.

## Existing Pattern Index

### approach-f-before-fm

- title: `Fm로 바로 가지 않고 F를 먼저 스치기`
- core move: `F/A` before `Fm7`
- target chord: `Fm7`
- duplicate keys: `F/A-Fm7`, `A-Ab`, `same-root-major-to-minor`, `approach-ii`
- voice motion: `A -> Ab`
- effect: bright opening immediately turns darker
- terms: `slash-chord`, `borrowed-chord`, `voice-leading`

### tonicize-minor-two

- title: `ii를 잠깐 중심처럼 세우기`
- core move: `Bbm7 -> C7 -> Fm7`
- target chord: `Fm7`
- duplicate keys: `Bbm7-C7-Fm7`, `iv-V-i-of-ii`, `tonicize-ii`
- bass motion: `Bb -> C -> F`
- effect: ii briefly becomes a local arrival
- terms: `tonicization`, `secondary-dominant`, `predominant`

### minor-iv-six-to-tonic

- title: `토닉으로 돌아가기 전에 마이너 iv6를 지나가기`
- core move: `Abm6 -> Eb/G` or `Fm6 -> C/E`
- target chord: tonic first inversion
- duplicate keys: `ivm6-I6`, `minor-iv-six`, `chromatic-bass-to-tonic`
- effect: a shadow appears just before the tonic settles
- terms: `borrowed-chord`, `voice-leading`, `chromatic-bass`

### chromatic-bass-slash-chords

- title: `분수코드로 베이스를 반음씩 내리기`
- core move: slash chords chosen to create chromatic bass motion
- duplicate keys: `slash-chord-bass-line`, `chromatic-bass`
- effect: complex chord changes feel inevitable because the bass line is simple
- terms: `slash-chord`, `voice-leading`, `chromatic-bass`, `common-tone`

### sus-dominant-door

- title: `서스 도미넌트로 새 조성의 문 열기`
- core move: `Bb9sus4 -> EbM7`, `Eb9sus4 -> AbM7`
- target chord: new tonic or new section opening chord
- duplicate keys: `9sus4-to-tonic`, `dominant-sus-modulation`
- effect: opens a new key area without a hard classical cadence
- terms: `tonicization`, `voice-leading`

### dominant-ninth-color

- title: `기본 도미넌트에 9음을 얹어 긴장을 부드럽게 넓히기`
- core move: `Bb7` becomes `Bb9`
- target chord: usually `Eb` or an Eb-family tonic
- duplicate keys: `Bb7-Bb9`, `dominant-ninth`, `add-9-to-dominant`
- effect: direct dominant becomes wider, warmer, and more soul/jazz influenced
- terms: `voice-leading`
