- Visual hierarchy and emphasis
- Spacing, alignment, and layout rhythm
- Typography, contrast, and readability
- Component consistency with nearby product patterns
- Responsive behavior at desktop and mobile sizes
- Interactive clarity for buttons, inputs, menus, and links
- Accessibility cues such as labels, focus states, and error visibility
