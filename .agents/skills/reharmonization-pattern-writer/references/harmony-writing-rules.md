# Harmony Writing Rules

Write like a professional composer or arranger explaining a reusable move to another musician.

The content should answer:

- What ordinary progression is being changed?
- What chord or voice-leading move is inserted?
- What sound does it create?
- Why does it work?
- Where can I reuse it?

## Title Rules

Titles should describe the action and effect.

Good:

- `Fm로 바로 가지 않고 F를 먼저 스치기`
- `ii를 잠깐 중심처럼 세우기`
- `토닉으로 돌아가기 전에 마이너 iv6를 지나가기`
- `분수코드로 베이스를 반음씩 내리기`

Avoid:

- `이도 장화음 색채`
- `서브도미넌트 마이너 컬러`
- `세컨더리 도미넌트 패턴`

## Explanation Order

Use this order unless the user asks for a different one:

1. One-sentence summary
2. Original progression
3. Reharmonized progression
4. What changed
5. What it sounds like
6. Why it works
7. When to use it
8. Related terms
9. Similar patterns or variants

## Tone

Prefer natural Korean.

Good:

> Fm로 바로 가지 않고 F/A를 먼저 두면, A가 Ab로 내려가면서 밝게 열렸던 소리가 바로 어두워진다.

Avoid:

> 이도 장화음 색채가 이도 마이너 색채로 전환된다.

## Duplicate Rules

Before creating a new pattern, compare the new idea against existing entries by:

- same inserted chord or chord family
- same target chord
- same voice motion
- same bass motion
- same emotional effect
- same theoretical device
- same reharmonized slot in a progression

If two or more are the same, propose updating the existing pattern.
